개발 환경

IDE(통합 개발 환경) : Eclipse
배우는 과정에서 쓰였던 프로그램이었기 때문에 익숙해서 사용하였습니다.

소스 코드 편집기 : Visual Studio Code (퍼블리싱 목적)
JavaScript와 CSS 등을 활용하여 UI를 디자인하고 구현하여 사용자가 접근 가능하게 만드는 용도로 사용하였습니다.

개발 언어 : Java 21버전

프레임워크 : Spring Boot 3.2.0 version
톰캣이 내장되어있으며 의존성주입이 편리하여 사용하였습니다.
언어는 Java를 사용하였고 빌드는 Maven으로 하였습니다.

데이터베이스 : MySQL

데이터베이스 관리툴 : MySQL Workbench

DB 접근기술 : JPA
데이터베이스 작업을 효율적이고 객체 지향적인 방식으로 처리할 수 있고, 
CRUD를 반복적인 작업 없이 간단하게 작업할 수 있습니다.

프론트엔드 프레임워크 : React.js
